# Citizen-Grievance-Management-System-CGMS
Its a salesforce project

## Problem Statement
Municipal departments handle thousands of citizen complaints across services (water, electricity, roads, sanitation) using manual/disconnected systems. This causes delays, lack of accountability, and poor transparency. 

The project aims to build a Salesforce-based grievance management system to:
- Provide a citizen-facing portal for complaint submission
- Automate routing to departments/officers
- Enforce SLA-based monitoring & escalation
- Provide dashboards for government administrators

## Objectives
- Centralize complaint logging
- Automate assignment & escalation
- Improve transparency with citizen notifications
- Deliver dashboards for tracking KPIs
